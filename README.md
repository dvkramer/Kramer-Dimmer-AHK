# Kramer Dimmer

An AHK script (and compiled executable) to dim your screen. This is intended to be used on (my) OLED laptops whose minimum brightness is set way too high. It looks flawless, as if the screen's brightness was natively allowed to get this low.

This is fully compatible with the auto-hidden taskbar. It changes the gamma in Windows rather than using a dark overlay so the taskbar does not get confused and render behind your other windows.

# Usage
- Rshift + [ and rshift + ] keyboard shortcuts to raise and lower brightness

I personally recommend setting this program to auto-start on login/startup in the task scheduler or shell:startup
